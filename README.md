# Clasificador de Residuos con Inteligencia Artificial

Proyecto de Inteligencia Artificial para la detección y clasificación de residuos sólidos utilizando redes neuronales convolucionales y transfer learning con MobileNetV2.

## Descripción

Sistema capaz de clasificar imágenes de residuos en 6 categorías:
- Cartón (cardboard)
- Vidrio (glass)
- Metal (metal)
- Papel (paper)
- Plástico (plastic)
- Basura general (trash)

El modelo fue entrenado con el dataset TrashNet y alcanzó una precisión de 90.85% en entrenamiento y 75.40% en validación después de 15 épocas.
